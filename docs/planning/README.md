## ğŸ¯ Overview

**Svony Browser V7.0 Ultimate Edition** is a specialized Windows desktop application designed for Evony game analysis, traffic monitoring, and automation. Built on .NET 6.0 with CefSharp Chromium browser engine, it provides a comprehensive suite of tools for game enthusiasts and developers.

### What Makes Svony Browser Special?

| Feature | Description |
|---------|-------------|
| **Dual Browser Panels** | Run AutoEvony Bot and Evony Client side-by-side |

## ğŸ’» Development

### Prerequisites

- Visual Studio 2022 or VS Code
- .NET 6.0 SDK
- Node.js 18+ (for MCP servers)
- Git

### Building from Source

```bash
# Clone the repository
git clone https://github.com/Ghenghis/Svony-Browser.git
cd Svony-Browser

# Restore dependencies
dotnet restore

# Build the solution
dotnet build -c Release

# Run the application
dotnet run --project SvonyBrowser/SvonyBrowser.csproj
```

### Publishing

```bash
# Self-contained Windows x64
dotnet publish SvonyBrowser/SvonyBrowser.csproj \
  -c Release \
  -r win-x64 \
  --self-contained \
  -o artifacts/release

# Framework-dependent (portable)
dotnet publish SvonyBrowser/SvonyBrowser.csproj \
  -c Release \
  -r win-x64 \
  --self-contained false \
  -o artifacts/portable
```

## ğŸ”§ Troubleshooting

### Common Issues

#### CefSharp Initialization Failed
```
Error: Could not load file or assembly 'CefSharp.Core.Runtime'
```
**Solution:** Use the self-contained (win-x64) release which includes all dependencies.

#### Flash SWF Not Loading
```
Error: AutoEvony.swf Not Found
```
**Solution:** Configure SWF paths in Settings â†’ Flash SWF Files section.

#### Fiddler Proxy Not Detected
```
Warning: Fiddler proxy not detected. Traffic won't be captured.
```
**Solution:** Start Fiddler Classic before launching Svony Browser, or enable "Auto-start Fiddler" in Settings.

#### Co-Pilot Shows "Connecting..."
```
Status: Connecting...
```
**Solution:** Ensure LM Studio is running on localhost:1234 with a model loaded.
